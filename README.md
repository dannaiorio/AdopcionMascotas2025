# AdopcionMascotas

Proyecto desarrollado en ASP.NET Core MVC para gestionar un refugio de mascotas.
Permite mostrar animales disponibles para adopción, registrar solicitudes y administrar el estado de cada mascota desde un panel de administración.


- Funcionalidades principales

Usuarios del sitio:

Visualizar mascotas disponibles (filtradas por especie: perros o gatos).
Consultar mascotas ya adoptadas.
Completar un formulario de adopción para una mascota específica.
Completar  un formulario de contacto para consultas generales.

Administradores:

Iniciar sesión en el panel de administración(con usuario y contraseña).
Crear, editar y eliminar mascotas.
Cambiar el estado de cada mascota (Disponible, En proceso, Adoptada).
Ver y gestionar solicitudes de adopción recbidas.
Ver mensajes recibidos a través del formulario de contacto.


- Tecnologías utilizadas:

.NET 9.0 / ASP.NET Core MVC
Entity Framework Core (ORM)
SQL Server LocalDB
Bootstrap 5 y CSS personalizado
Razor Views
Visual Studio 2022
